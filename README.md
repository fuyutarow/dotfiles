# dotfiles

### setup
```shell
cat .bashrc > ~/.bashrc
cat .vimrc > ~/.vimrc
cat .gitconfig > ~/.gitconfig 
cat keyhac.py > keyhac/config.py
```

##### vim plugins install
```
mkdir -p ~/.vim/bundle
git clone https://github.com/gmarik/Vundle.vim.git ~/.vim/bundle/Vundle.vim
```
